Unidades de Medidas Absolutas

Também são chamadas de medidas fixas, não dependem do elemento pai, sempre terão o mesmo tamanho, independênte de onde estão inseridas.

PIXEL

O pixel é a menor unidade que pode ser exibida na tela do dispositivo, seja ele um celuar, um tablet, um notebook, etc.

Isso pode gerar um problema, se a altura ou largura de um elemento for maior que a quantidade máxima de pixels de um dispositivo, esse elemento ultrapassará a tela, devendo o usuário scrolar para ver todo o elemento.
